# Linux Web Deployment Project

## Project Description
This is my first project using Linux to deploy a website with Nginx.

## What I did
- Installed and configured Nginx
- Deployed a static website
- Managed Linux file system and permissions
- Used SSH to connect to the system

## Technologies
- Linux (Ubuntu 24.04)
- Nginx
- Git & GitHub

## How to Run
1. Install Nginx
2. Place index.html in /var/www/html
3. Start Nginx:
   sudo systemctl start nginx
4. Open browser: http://localhost
